# 📰 AI日报摘要

## 2026-07-04

**第一，AI 视频和算力正在成为独立的大资本赛道。** Kling AI 以 180 亿美元估值完成 30 亿美元融资，Meta 则计划将巨额 capex 产生的闲置算力对外出租。这两件事指向同一个趋势：AI 产业的价值重心正在从"模型能力"向"稀缺资源"转移——无论是视频生成所需的算力和数据，还是推理所需的数据中心容量。拥有这些资源的公司，正在获得比纯模型公司更稳定的议价能力。

**第二，AI 编码工具的安全边界正在被重新定义。** Cursor DuneSlide 漏洞表明，提示注入不再是让模型"说错话"的小问题，而是可以突破沙盒、控制开发者主机的系统级攻击向量。随着 coding agent 获得自动执行代码、提交 PR、连接 SaaS 的权限，其攻击面已经远超传统 IDE。企业需要把 AI 编码工具纳入核心安全治理框架，而不是当作普通生产力软件。

**第三，AI 转型正在进入"组织重构"的深水区。** Meta 的 8000 人裁员、Cisco 的 4000 人裁员与 9 万 agent 部署、美国 6 月就业数据中创纪录的 AI 归因裁员，共同描绘出一幅图景：AI 对企业劳动力市场的影响已经从"局部优化"走向"结构性替代"。对科技公司而言，最大的风险可能不再是模型能力不足，而是组织转型过程中的文化崩溃、人才流失和社会反弹。未来 6-12 个月，"AI 转型治理"将成为董事会层面的核心议题。
