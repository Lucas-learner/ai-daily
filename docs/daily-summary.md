# 📰 AI日报摘要

> 每日精选洞察，完整日报请查看对应月份文件。

## 2026-07-21

1. **算力瓶颈第一次同时出现在需求侧与效率侧。** Kimi 因请求量逼近集群极限暂停新订阅——中国 lab 首次因 C 端需求过旺主动刹车；同一天，Alphabet 被曝以 6-10 倍能效的 Frozen v2 押注 2028 年，Anthropic 则经一次 GitHub 泄密被确认为 AMD 客户。三件事指向同一转折：竞争的计量单位正从"买了多少 GPU"变成"每瓦、每美元产出多少 token"。东吴证券关于"AI 交易从资本开支转向效率与 ROI"的研报此时出现并非巧合——本周 Alphabet/Intel/Tesla 财报与 AMD Advancing AI 大会，将是"效率叙事"对"扩张叙事"的第一次正面检验。
2. **欧洲用裁判、用户、出资人三重身份同时入场。** DMA 强制 Android 向竞品 AI 助手开放系统权限（Google 合规换时间、Apple 延迟对抗）、欧洲议会自建 EPGenAI Hub 供议员使用、Current AI 用 4 亿美元公共资金建开放基础设施——与美国"出口管制+自愿框架"的单一路径相比，欧洲正把监管、采购与公共投资组合成一套完整的生态工具箱。短期看是给 OpenAI/Anthropic 送分发红利，长期看是在为"没有操作系统的模型公司"与"没有模型公司的操作系统"之间重新划界。
3. **中国 AI 进入"履约周"，资本市场给出定价锚。** WAIC 闭幕留下 203.6 亿元意向采购与 409 亿元签约的实收数据；长鑫科技 579 亿元科创板最大 IPO 缴款、243 倍申购，把"AI 需要存储"的故事变成 A 股定价锚。接下来一周全是硬性履约节点：7 月 24 日 DeepSeek 旧 API 硬停用、V4 正式版临近，7 月 27 日 K3 权重开放与长鑫挂牌同日而至——发布周赚的是眼球，履约周赚的是信用。Musk 的"Impressive"与 WSJ 的"We're coming for you"则共同说明：美国舆论对中国 AI 的态度，已从惊讶切换到正视。
