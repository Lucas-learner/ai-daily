## 2026-07-20

1. **Frontier 定价进入"算力配给制"时代。** Anthropic 用一条 $100/月的分界线，把最强模型从"订阅平权"变成"高档位专属资产"，中间档用户被推向按量计费；同一周里，微软用自研 MAI 消减外采成本、阿里用一折推广价抢负载、Sonnet 5 以 $2/$10 引流价托底——厂商们在"frontier 溢价"与"规模化降本"之间双向挤压，被挤出的是"平价用最强模型"这个曾经默认存在的中间地带。对用户的实际建议从未如此清晰：日常工作负载锚定中端模型，frontier 只花在真正需要它的任务上。
2. **编码 Agent 的竞争从"会话智能"升级为"云基础设施"。** OpenAI 买 Ona、SpaceX 买 Cursor、Anthropic 推 Ode——同一月三笔交易指向同一判断：Agent 的价值不再取决于单次对话多聪明，而取决于能否 7×24 小时在云端持续干活、并嵌入企业的权限与审计体系。Grok Build 上传事件与 Sol 删文件根因报告则从反面标注了底线：没有正式事故响应和权限治理的 Agent，能力越强， blast radius 越大。
3. **中国模型用一周走完"叙事—产品—资本"的完整闭环。** WAIC 会期内：K3 以 2.8T 开源权重定能力锚点、Qwen3.8 以 2.4T 与一折定价抢份额、DeepSeek 以 3250 亿+ 估值完成资本锚定，SAIL 奖与超节点上云补齐基础设施叙事。下周进入执行验证期：7 月 24 日 DeepSeek 旧 API 停用、7 月 27 日 K3 权重落地、Qwen3.8 开源承诺待兑现——这三件事的履约质量，将决定"中国开源 frontier"是周期性的发布热潮，还是可持续的供给能力。
